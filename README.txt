Planificador de Puntos Familiar — Versión Mobile Friendly
=============================================================

Mejoras móviles:
- Diseño mobile-first, tabs pegajosas (sticky) y botones grandes.
- Inputs de tiempo/fecha optimizados para teclado móvil.
- Tablas con contenedor desplazable horizontal y etiquetas responsivas.
- Todo funciona offline (archivo estático).

Cómo publicar (GitHub Pages)
----------------------------
1) Sube `index.html`, `styles.css`, `app.js` a un repositorio público.
2) En Settings → Pages: Deploy from a branch → main /(root).
3) Abre `https://<usuario>.github.io/<repo>/` en el teléfono y añade a la pantalla de inicio.

Tarifas y reglas
----------------
Se mantienen exactamente las franjas y reglas acordadas.

Cómo publicar en GitHub Pages
-----------------------------
1) Sube `index.html`, `styles.css`, `app.js` a un repo público.
2) Settings → Pages → Deploy from a branch (main / root).
3) Abre `https://<usuario>.github.io/<repo>/` en tu celular.

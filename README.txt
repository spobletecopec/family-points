Planificador de Puntos Familiar — v2.1 (parches de botones)
================================================================

Arreglos:
- Botones con `type="button"` para evitar envíos de formulario no deseados.
- Listeners con comprobaciones nulas y manejo defensivo.
- Delegación de clics para eliminar robusta (usa `closest()` y `dataset` del botón).
- Tap targets más grandes.

Instalación rápida
------------------
1) Descomprime el zip.
2) Abre `index.html` en tu navegador o súbelo a GitHub Pages.
